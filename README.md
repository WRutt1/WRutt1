## 🚀 Key Projects

### 🌾 AgroTech Monitoring System `In Progress`
> **Concept:** IoT-driven backend for real-time temperature and humidity monitoring in grain elevators.
* **Tech Stack:** `Go` (for high performance), `MQTT`, `PostgreSQL`, `Nginx`.
* **Impact:** Prevents grain spoilage and automates manual routine tasks on real agricultural facilities.
